Example: EPS Calculation Script


def earnings_per_share(net_income, preferred_dividends, weighted_avg_shares):
    """
    Basic EPS formula:
    EPS = (Net Income - Preferred Dividends) / Weighted Avg. Shares
    """
    return (net_income - preferred_dividends) / weighted_avg_shares

# Example values (in millions)
net_income = 1200
preferred_dividends = 50
weighted_avg_shares = 75

eps = earnings_per_share(net_income, preferred_dividends, weighted_avg_shares)
print(f"Quarterly EPS: ₹{eps:.2f}")

Key Financial Formulas

Return on Equity (ROE):
\[ ROE = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

Present Value (PV) of cash flow:
\[ PV = \frac{CF_t}{(1 + r)^t} \]

Earnings Per Share (EPS):
\[ EPS = \frac{\text{Net Income} - \text{Preferred Dividends}} {\text{Weighted Average Shares Outstanding}} \]
